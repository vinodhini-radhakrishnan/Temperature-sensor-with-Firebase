# Temperature-sensor-with-Firebase
 In this project it is described to get and push temperature data from dht11 using nodemcu to firebase.
## Components Used
    DHT11
    NodeMCU (ESP8266)
    Jumpers
 ## Circuit connection
    connect VCC -5V
    data-D4
    gnd -gnd
